# Spy-Eye
Spy-Eye is a tool that generates disguised links appearing as normal content, such as videos or chat pages, and is claimed to attempt access to device features like the front camera to capture images when the link is opened, often without clear user awareness. 
Only For Educational and Testing Purposes... Not to harm Someone.
